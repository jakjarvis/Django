Stock App - Forked version

This app allows the user to input their stock transactions, and will return information about their portfolio. It is build in Django, but uses functionality from Pandas to retrieve and manipulate stock data in the back-end and Plotly/Dash to present graphical information in an interactive and user friendly way. The styling is largely performed using Bootstrap CSS.
